* SRFI 34: Exception Handling for Programs

** by Richard Kelsey and Michael Sperber

This repository hosts [[http://srfi.schemers.org/srfi-34/][SRFI 34]]: Exception Handling for Programs, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

This SRFI is in /final/ status.

The full documentation for this SRFI can be found in the [[http://srfi.schemers.org/srfi-34/srfi-34.html][SRFI Document]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-34/][join the SRFI-34 mailing list]] and send your message there.

Thank you.
